# ControleDeMedicamentos

![](https://i.imgur.com/vnJP6Qk.gif)

## Descrição
Este é um sistema simples de controle de medicamentos que permite ao usuário gerenciar diferentes entidades relacionadas ao ambiente hospitalar ou farmacêutico, como fornecedores, pacientes, medicamentos, funcionários e prescrições.

## Funcionalidades
O programa oferece ao usuário sete opções principais de gerenciamento:

- *Fornecedores*
- *Pacientes*
- *Medicamentos*
- *Funcionários*
- *Prescrições Médicas*
- *Requisições de Saída*
- *Requisições de Entrada*

Após escolher uma dessas categorias, o usuário poderá realizar as seguintes ações:

### 1. Cadastrar
- O programa solicitará os dados necessários para o cadastro da opção selecionada.
- Caso algum dado obrigatório não seja informado, o cadastro será reiniciado.

### 2. Editar
- Permite modificar os dados de registros previamente cadastrados.

### 3. Excluir
- Remove um cadastro da memória com base no ID informado pelo usuário.

### 4. Visualizar
- Exibe todos os registros cadastrados na memória para a opção base selecionada.

## Como Utilizar
1. Execute o programa.
2. Escolha uma das sete opções base disponíveis.
3. Escolha a ação desejada: Cadastrar, Editar, Excluir ou Visualizar.
4. Siga as instruções apresentadas na tela.
